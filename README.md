## Hvordan i tester det selv.

Tænker godt i ved hvordan man tester det, men hvis i er i twivl er der en lille guide nede for.

```bash

git clone https://github.com/taxidriver2192/pentia-v2.git

cd pentia-v2

npm i

npm run dev

```
